## Pac-Man Game

A browser-based Pac-Man game built using HTML, CSS, and JavaScript.
This project recreates the classic arcade gameplay where the player navigates through a maze, collects dots, and avoids ghosts.

## Game Description

The player controls Pac-Man using the keyboard arrow keys and moves through a maze to collect dots and power pellets. Ghosts move inside the maze and try to catch the player.

The objective is to collect all dots while avoiding the ghosts.

## Features

Maze-based gameplay

Pac-Man movement using arrow keys

Ghost enemies with automatic movement

Score tracking system

Power pellets

Game Over and Win conditions

## Technologies Used

HTML – Creates the structure of the game

CSS – Styles the maze, characters, and animations

JavaScript – Handles game logic and interactions

## Concepts Used

This project demonstrates the following programming concepts:

Grid-based layout

Keyboard event handling

Collision detection

Game state management

Basic enemy AI logic

DOM manipulation
