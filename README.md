# Case Study Projects

Author: Ved Patil
Course: B.Tech CSE

This repository contains two case study projects developed as part of my coursework.

## 1. Telemedicine Backend System (C++)

A command-line telemedicine system developed using C++.
It demonstrates Object-Oriented Programming concepts and file handling.

**Main features:**

* Register doctors and patients
* Schedule consultations
* Store consultation records in a file
* Prevent duplicate IDs

**Concepts used:**

* OOP (Classes, Inheritance, Polymorphism)
* Vectors
* File handling
* Exception handling

## 2. Pacman Game (HTML, CSS, JavaScript)

A browser-based Pacman game created using web technologies.

**Main features:**

* Maze-based gameplay
* Pacman movement using arrow keys
* Ghost enemies
* Collision detection
* Score system

**Technologies used:**

* HTML
* CSS
* JavaScript

#Purpose

These projects demonstrate the practical application of programming concepts using C++ and web technologies such as HTML, CSS, and JavaScript.
